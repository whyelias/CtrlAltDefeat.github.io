02 — Development Log

Project Timeline

Chronological updates on what we've built, broken, and fixed.

April 1, 2025
Planning Design
Project Kickoff & Initial Architecture
Posted by Alex M.

We held our first full team meeting and finalized the tech stack. The project will use a three-tier MVC architecture — HTML/CSS/JS frontend, Java servlet backend, and a MySQL database.

Key decisions made:

  • Set up the GitHub repository and branch strategy (main / dev / feature branches)
  • Created the Trello board and distributed initial use cases
  • Agreed on a shared coding style guide and commit message format
  • Sketched the first wireframes for the hotel booking flow

Next sprint goal: get the database schema drafted and the login flow working end-to-end.

April 8, 2025
Backend Database
Database Schema & Auth Endpoints Complete
Posted by Taylor K. & Alex M.

The full ER diagram is finalized and the schema has been implemented in MySQL. Tables include: users, roles, rooms, reservations, products, and orders.

Authentication servlets are live in the dev environment — register, login, logout, and session management all tested via Postman.

April 15, 2025
Frontend Design
UI Component Library & Guest Booking Page
Posted by Sam R. & Casey W.

Built out the shared component library: buttons, form inputs, cards, and modal dialogs. All components follow WCAG 2.1 AA contrast ratios.

The guest-facing booking page is now complete — users can browse available rooms filtered by date range, room type, and guest count.

03 — The Project

Hotel & Shop System

A brief overview of what we're actually building.

What We're Building

A web-based hotel and retail shop management system that allows guests to browse and book rooms, and customers to purchase shop products — all through a unified interface backed by a shared database.

An admin panel allows staff to manage room availability, process check-ins, handle inventory, and view sales and reservation reports.

Tech Stack & Scope

  • Frontend: HTML, CSS, JavaScript
  • Backend: Java Servlets (or your framework here)
  • Database: MySQL
  • Version Control: Git / GitHub
  • Project Management: Trello
  • Deployment: GitHub Pages (frontend) + local/hosted backend
